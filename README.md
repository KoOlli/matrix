# matrix

Написать свою библиотеку matrix.h на С.

Реализовать функции:
- создание матрицы
- очистка матриц
- сравнение матриц
- сложение и вычитание матриц
- умножение матрицы на число
- умножение двух матриц
- транспонирование матрицы
- минор матрицы
- матрица алгебраических дополнений
- определитель матрицы
- обратная матрица
